# XPay Labs Checkout

English | [中文](README.zh.md)

XPay Labs cryptocurrency payment checkout page — a Vue 3 + TypeScript payment gateway interface.

## Features

- Multi-language support (EN, ZH-CN, ZH-TW)
- Multi-chain cryptocurrency payments (Ethereum, Bitcoin, BSC, Polygon, Tron)
- Real-time payment status polling
- Responsive design

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:5173/?orderId=ORDER_123&sign=your_signature&lang=en`

## Tech Stack

- Vue 3 + TypeScript + Vite
- Pinia (state management)
- Vue Router
- Axios (HTTP client)
- QRCode (payment QR generation)
